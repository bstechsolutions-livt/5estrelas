Instruções — Integração EASYTECH (Hub 5 Estrelas)

Para: Luan  |  Data: 12/07/2026  |  Preparado por: Easytech

Objetivo: Liberar consulta de Fornecedor, Filial e Centro de Custo via webservice para o sistema integrado EASYTECH, em cada empresa do grupo.

Já feito e validado: Empresa 2 (5 ESTRELAS). Replicar o mesmo nas demais.
Não precisa fazer: Empresa 1 (Modelo) e Empresa 12.

1. Conferência única (antes de começar)

Tela: Cadastro de Sistemas Integrados

Deve existir o registro:

  • Sigla: EASYTECH
  • Tipo: 99 - Outros
  • Situação: A - Ativo

Se já existir, não alterar.

2. Rotina por empresa

Repetir os passos abaixo para cada empresa da lista (seção 4).

1 Selecionar a empresa

Tela: Seleção de Empresa

  1. Abrir a tela Seleção de Empresa
  2. Selecionar a empresa na lista
  3. Clicar Ok
  4. Confirmar no canto superior direito que aparece a empresa correta (ex.: 0003 SERV APOIO)

2 Ativar tipos de informação

Tela: Configuração de Tipos de Informação

Menu: Cadastros → Integrações → Configuração de Tipos de Informação

  1. Sistema: EASYTECH
  2. Filial: 1
  3. Informação: deixar em branco
  4. Clicar Mostrar
  5. Na grade, mudar Situação para A - Ativo
  6. Na confirmação, escolher Todos os Itens
  7. Marcar os registros
  8. Clicar Processar

Se a empresa tiver mais de uma filial: usar Duplicar para replicar da Filial 1 para as demais.

3 Webservice: Fornecedor

Tela: Configuração de web services

Menu: Cadastros → Integrações → Configuração de web services

  1. Clicar Tab no campo Código (gera código novo — não reutilizar)
  2. Empresa: a empresa logada
  3. Filial: 1
  4. Sistema: EASYTECH
  5. Web service: Cadastro - Fornecedor
  6. Porta: Consultar Geral
  7. Marcar todos os campos em "Campos exportados pelo web service"
  8. Clicar Processar

4 Webservice: Filial

Tela: Configuração de web services (mesma tela)

  1. Código: Tab (novo código)
  2. Empresa / Filial / Sistema: mesmos valores
  3. Web service: Cadastro - Filial
  4. Porta: Consultar Geral
  5. Marcar todos os campos → Processar

5 Webservice: Centro de Custo

Tela: Configuração de web services (mesma tela)

  1. Código: Tab (novo código)
  2. Empresa / Filial / Sistema: mesmos valores
  3. Web service: Cadastro - Centro de Custo
    (Se o nome for diferente, ex.: Gerencial - Centro de Custo, usar o que tiver porta Consultar Geral)
  4. Porta: Consultar Geral
  5. Marcar todos os campos → Processar

6 Plano de Contas — pular

Não configurar. Esse webservice não existe no ambiente. O hub usa plano de contas provisório a partir dos títulos de CP/CR.

7 Outras filiais (somente se existir)

Se a empresa tiver Filial 2, 3, etc.:

  • Em Configuração de web services: após cada webservice na Filial 1, usar Duplicar
  • Em Configuração de Tipos de Informação: idem — Duplicar

Se só tem Filial 1 (matriz): não precisa duplicar.

3. Resumo por empresa

Seleção de Empresa → Ok Configuração de Tipos de Informação → Processar (1×) Configuração de web services → Processar (3×): 1. Cadastro - Fornecedor / Consultar Geral 2. Cadastro - Filial / Consultar Geral 3. Cadastro - Centro de Custo / Consultar Geral Total: 4 Processar por empresa (1 + 3)
